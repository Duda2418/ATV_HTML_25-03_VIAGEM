-Descrição

ExploreMundo é um site de viagens que apresenta destinos turísticos pelo mundo, fornece informações sobre a empresa e oferece um formulário de contato. O objetivo é criar uma experiência interativa, organizada e visualmente atraente para os usuários que desejam explorar o mundo.

 -Funcionalidades
Página principal com menu de navegação para as seções: Início, Destinos, Sobre, Contato
Seção Destinos:
Cards de destinos turísticos com imagem, título e descrição
Layout responsivo em 3 colunas
Seção Sobre Nós:
Informações sobre a empresa
Botão laranja padronizado “Leia Mais”
Seção Contato:
Formulário de contato com campos de nome, e-mail e mensagem
Informações de localização, telefone e e-mail
Layout flexível, centralizado e responsivo
Menu superior com links de âncora que rolam suavemente para cada seção
Botões estilizados com cores consistentes e hover interativo
Layout moderno, responsivo e centrado
